# Custom Instructions (Addon Mode)

This file adds ADDITIONAL instructions to the default system prompt.
The default prompt is preserved, and your instructions are appended.

To use:
1. Copy this file to `system-prompt-addon.txt` in the same directory
2. Add your custom instructions below
3. The extension will automatically append them to the default prompt
